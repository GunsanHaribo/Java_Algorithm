2. 종료

### DP
mem = 특정 시점의 상태 기록 

### DFS & BFS 
그래프 그리기
1. 스택(DFS) -> 깊이제한, 최단불가
   1. 자식노드 하나로 계속 우선 전이
   2. 방문체크
2. 큐(BFS) -> 최단거리탐 색 
   1. 모든 자식노드로 우선 전이
   2. 방문체크

1. 비트연산 (&, |, ^,~, <<,>>)

3. 가능한 경우의수 찾기(x1, x2, F)
4. 동적객체(손, 클래스)
