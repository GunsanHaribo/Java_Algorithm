# Two Pointers

- 배열과 같은 선형 자료구조에서 두 개의 포인터를 사용하는 알고리즘
- 원래는 이중 반복문으로 $O(n^2)$에 처리하던 작업을, 포인터 2개의 움직임을 통해 $O(n)$에 해결할 수 있음
