### 스택 == 짝찾기 but how? 
푸는 순서 
1) 짝 정의 = stack에 있는 수 보다 작은 수 찾기 
2) 짝을 찾는 방법 - 짝정의에 맞게 반복문과 stack의 적절한 활용 
   1) for 문을 통해서 순회를 하면서 현재 stack에 값을 넣는다
   2) for문 돌면서 현재값을 통해서 stack하고 맞는 짝을 찾는다
